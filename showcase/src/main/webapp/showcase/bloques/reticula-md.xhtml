Retícula col-md-*

Elemento para representar una maqueta de un sitio web para dispositivos medianos.

 EJEMPLO col-md-12 col-md-6 col-md-6 col-md-4 col-md-4 col-md-4 col-md-3 col-md-3 col-md-3 col-md-3 col-md-2 col-md-2 col-md-2 col-md-2 col-md-2 col-md-2

Esta clase esta diseñada para dispositivos medianos Escritorios (≥992px).

Basado en 12 columnas.

Se recomienda envolver las clases .col-**-* con la clase .row.

Permite anidado entre clases .col-xs-* .col-sm-* .col-lg-* para una mejor visualización en cualquier resolución.
