Retícula col-xs-*

Elemento para representar una maqueta de un sitio web para dispositivos muy pequeños.

 EJEMPLO col-xs-12 col-xs-6 col-xs-6 col-xs-4 col-xs-4 col-xs-4 col-xs-3 col-xs-3 col-xs-3 col-xs-3 col-xs-2 col-xs-2 col-xs-2 col-xs-2 col-xs-2 col-xs-2

Esta clase esta diseñada para dispositivos extra pequeños Phones (<768px).

Basado en 12 columnas.

Se recomienda envolver las clases .col-**-* con la clase .row.

Permite anidado entre clases .col-sm-* .col-md-* .col-lg-* para una mejor visualización en cualquier resolución.
