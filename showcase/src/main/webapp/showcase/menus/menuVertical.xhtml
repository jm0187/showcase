Menú Vertical

En esta sección se mostrara como crear un menú vertical haciendo uso de Bootstrap.

EJEMPLO
  • Home
  • Menu 1
    • Submenu 1-1
    • Submenu 1-2
    • Submenu 1-3
  • Menu 2
  • Menu 3

Para generar un menú horizontal se hace uso de las clases de Bootstrap nav y nav-pills nav-stacked .

  • XHTML
<ul class="nav nav-pills nav-stacked">
	<li class="active"><h:outputLink value="#">Home</h:outputLink></li>
	<li class="dropdown"><a class="dropdown-toggle"
		data-toggle="dropdown" href="#">Menu 1 <span class="caret"></span></a>
		<ul class="dropdown-menu">
			<li><h:outputLink value="#">Submenu 1-1</h:outputLink></li>
			<li><h:outputLink value="#">Submenu 1-2</h:outputLink></li>
			<li><h:outputLink value="#">Submenu 1-3</h:outputLink></li>
		</ul></li>
	<li><h:outputLink value="#">Menu 2</h:outputLink></li>
	<li><h:outputLink value="#">Menu 3</h:outputLink></li>
</ul>
