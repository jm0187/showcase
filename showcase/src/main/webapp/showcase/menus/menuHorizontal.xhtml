Menú Horizontal

En esta sección se mostrara como crear un menú horizontal haciendo uso de Bootstrap.

EJEMPLO
  • Home
  • Page 1
  • Page 2
  • Page 3

Para generar un menú horizontal se hace uso de las clases de Bootstrap nav y nav-pills .

  • XHTML
<ul class="nav nav-pills">
	<li class="active"><h:outputLink value="#">Home</h:outputLink></li>
	<li><h:outputLink value="#">Page 1</h:outputLink></li>
	<li><h:outputLink value="#">Page 2</h:outputLink></li>
	<li><h:outputLink value="#">Page 3</h:outputLink></li>
</ul>
