Opciones de Botones

A continuación se muestran los diferentes estilos que se le pueden dar a un botón

EJEMPLO

Para cambiar el tamaño del botón es necesario agregar el atributo styleClass="btn btn-default" con el tamaño requerido.

Para una mejor visualización de los botones es necesario que se envuelvan dentro de contenedor de tipo h:panelGroup con el atributo styleClass="form-group" .

  • XHTML
					
<h:panelGroup styleClass="col-md-2" layout="block">
	<h:panelGroup styleClass="form-group" layout="block">
		<p:commandButton value="Básico"
			styleClass="btn btn-default form-control" />
	</h:panelGroup>
</h:panelGroup>
