Agrupación de elementos

En esta sección se mostrara como agrupar elementos dentro de un contenedor, en el caso de una etiqueta y un input es mejor agruparlos para darles una mejor presentación.

EJEMPLO

Para agrupar elementos en necesario agregar un h:panelGroup con la propiedad styleClass="form-group" y cada uno de los controles agrupados deben de tener la propiedad styleClass="form-control" .Como se muestra en el siguiente ejemplo:

  • XHTML
<h:panelGroup styleClass="form-group" layout="block">
	<h:outputLabel value="Campo de Texto"/>
	<h:inputText styleClass="form-control"/>	
</h:panelGroup>
