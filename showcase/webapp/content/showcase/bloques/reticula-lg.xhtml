Retícula col-lg-*

Elemento para representar una maqueta de un sitio web para dispositivos grandes.

 EJEMPLO col-lg-12 col-lg-6 col-lg-6 col-lg-4 col-lg-4 col-lg-4 col-lg-3 col-lg-3 col-lg-3 col-lg-3 col-lg-2 col-lg-2 col-lg-2 col-lg-2 col-lg-2 col-lg-2

Esta clase esta diseñada para dispositivos grandes Escritorios (≥1200px).

Basado en 12 columnas.

Se recomienda envolver las clases .col-**-* con la clase .row.

Permite anidado entre clases .col-xs-* .col-sm-* .col-md-* para una mejor visualización en cualquier resolución.
