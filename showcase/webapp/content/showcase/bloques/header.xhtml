Header

Elemento para colocar encabezados en las páginas.

 EJEMPLO

Infotec.com.mx

Para el uso del elemento header es necesario generar un panel usando panelGroup y agregar los atributos: styleClass="col-sm-* col-md-*" layout="block" donde los * representan el número de columnas a ocupar, el mismo código aplica para agregar el dominio de la empresa.
  • XHTML
<header>
	<h:panelGroup styleClass="row" layout="block">
		<h:panelGroup styleClass="col-sm-6 col-md-6" layout="block">
			<h:link outcome="bloques">
				<h:graphicImage value="/resources/images/template/infotec.png"/>
			</h:link>
		</h:panelGroup>
		<h:panelGroup styleClass="col-sm-6 col-md-6" layout="block">
			<h4 class="hidden-xs">Infotec.com.mx</h4>
		</h:panelGroup>
	</h:panelGroup>
</header>
