Ventana Modal

En esta sección se mostrara como crear una ventana modal utilizando Bootstrap

EJEMPLO

Título del Modal

"Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua."

Para crear la ventana modal se hace uso de la clase modal de Bootstrap.

Para mostrar la ventana modal se utiliza la propiedad onclick="$('#myModal').modal('show');"

Para ocultar la ventana modal se utiliza la propiedad onclick="$('#myModal').modal('hide');"

  • XHTML
<h:panelGroup styleClass="col-md-4" layout="block">
	<p:commandButton value="Mostrar"
		onclick="$('#myModal').modal('show');" process="@this"
		styleClass="btn btn-primary form-control" />
</h:panelGroup>

<div class="modal fade" id="myModal" tabindex="-1" role="dialog"
	aria-labelledby="myModalLabel" aria-hidden="true">
	<div class="modal-dialog">
		<div class="modal-content">
			<div class="modal-header">
				 <button type="button" class="close" data-dismiss="modal" aria-hidden="true">
				 	<span class="glyphicon glyphicon-remove-sign"></span>
				</button>
				<h4 class="modal-title">Título del Modal</h4>
			</div>
			<div class="modal-body">
				<p>"Lorem ipsum dolor sit amet..."</p>
			</div>
			<div class="modal-footer">
				<p:commandButton value="Cerrar"
					onclick="$('#myModal').modal('hide');" process="@this"
					styleClass="btn btn-primary" />
			</div>
		</div>
		<!-- /.modal-content -->
	</div>
	<!-- /.modal-dialog -->
</div>
<!-- /.modal -->
